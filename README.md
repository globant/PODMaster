PODMaster
=========

PODMaster Dashboard Application

###How to install

  * git clone https://github.com/globant/PODMaster.git
  * npm install
  * bower install
  * grunt server &
  * go to http://localhost:9001/#index/http://localhost:9001/js/app/modules/dashboard/projected_schedule.json
